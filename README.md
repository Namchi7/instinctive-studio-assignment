# Instinctive Studio - Full Stack Assignment

### How to run locally

- Clone this repository
- Go to the repository directory
- Run command to install dependency packages:

```
npm install
```

- After installation run the development project:

```
npm run dev
```

### Tech Used

- Next.js
- React.js
- TypeScript
- Redux-Toolkit
- Shadcn/ui
- Prisma
